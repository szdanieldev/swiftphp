### Project Structure

```
/app
  /Controllers
  /Models
  /Views
/config
/core
/localization
/public
/routes
/storage
/docs
```

## app/

Main application logic.

- Controllers: Handle incoming requests
- Models: Database and business logic
- Views: HTML rendering

## core/

Framework engine (Router, Request, Response, Auth, Localization loader).

## config/

Application configuration files.

## localization/

Language files grouped by locale.

## public/

Entry point (`index.php`) and public assets.

## routes/

Route definitions.

## storage/

Logs and session data.
